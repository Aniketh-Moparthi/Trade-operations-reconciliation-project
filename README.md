# Trade Operations Reconciliation & Break Analysis

## Project Overview
Built an Excel-based trade operations workflow simulating post-trade reconciliation between internal and broker trade records across multiple asset classes.

## Key Functions
- Trade reconciliation
- Break identification
- Exception reporting
- Quantity and price validation
- Dashboard reporting

## Asset Classes Covered
- Equity
- ETF
- Fixed Income
- Derivatives

## Tools Used
- Microsoft Excel
- VLOOKUP
- IF Statements
- Pivot Tables
- Dashboard Reporting

## Key Breaks Identified
- Quantity mismatches
- Price mismatches
- Missing trades

## Workflow
Internal Trades → Broker Trades → Reconciliation → Break Report → Dashboard
